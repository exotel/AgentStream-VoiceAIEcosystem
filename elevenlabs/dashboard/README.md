# ElevenLabs — Dashboard (manual setup)

Use this track to get things done **through the ElevenLabs web dashboard** without writing code first.

## Suggested steps

1. **Sign up / log in** at [elevenlabs.io](https://elevenlabs.io/).
2. **Subscription & limits** — confirm plan, character quotas, and concurrency for your use case.
3. **Voices** — browse **Voice Library**, create **Voice Design** / **Instant Voice Clone** / **Professional Voice Clone** as needed; note **Voice ID** for later API use.
4. **Projects / Studio** (if using content workflows) — create projects, scripts, and exports as the product allows.
5. **API keys** — under profile/settings, create a key when you are ready to connect the API track in [`../api/`](../api/).
6. **History & usage** — monitor generation history and usage to stay within limits.

## When to switch to the API

After voices and settings look right in the UI, use the [`api/`](../api/) folder to automate TTS, voice selection, and integrations.
